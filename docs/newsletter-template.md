# Newsletter template

Reusable broadcast template for sudomakevibe.com newsletter sends.
One email every two weeks, paired with each new blog post.

---

## Subject line

```
new post: {post title or shortened version}
```

Keep under ~40 characters for mobile preview. Use lowercase to match site voice.

---

## Body

```
$ sudo get updates
longer reads | deeper breaths | sudo make calm
```

---

## {post title}

{post description from frontmatter}

[**Read on the site →**](https://sudomakevibe.com/blog/{post-slug})

---

### A note from me

{2-3 sentences. Structure: origin → what surprised me → what is next. Add context the post itself does not need to carry. If nothing comes to mind, leave this section out — better to skip than to fill.}

— Farooq

## How to use it

1. Copy the body structure into a new ConvertKit broadcast
2. Replace the four placeholders: subject line, post title, post description, post slug
3. Write the personal note — or delete the entire section if nothing genuine comes to mind
4. Send

## Voice rules

- All lowercase in subject lines, headings stay sentence case
- No contractions in the personal note (matches site voice)
- Personal note is optional — never force it
- ConvertKit auto-injects the unsubscribe link, no need to add one manually
